# Epic starter kit

Стартовый репозиторий для моих студентов.

**ВНИМАНИЕ!** Не клонировать! Форкните себе этот репозиторий и клонируйте свой форк!

Установка: `npm i`.

Запуск: `npm start`.

Остановка: <kbd>Ctrl + C</kbd>
